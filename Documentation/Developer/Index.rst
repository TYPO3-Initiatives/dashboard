Extend predefined widgets
-------------------------

The extension ships a list of predefined widgets to ensure a consistent look of tables, graphs and data.

The shipped widgets are in the folder `Classes/Widgets`.


-------------------------------
Make a completely custom widget
-------------------------------

Each and every has to implement the `WidgetInterface`, for most cases it's recommend to extend one of the
`Abstract(.*)Widget` implementations.
